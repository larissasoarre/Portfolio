Dublin Blog Template

Project Overview

This template, created with personal or professional bloggers in mind, was developed using the latest HTML, CSS, and JavaScript to deliver a stunning and highly functional design, ensuring seamless integration and an unbeatable user experience. The project counts with six unique page designs, including a custom 404 error page, making it a versatile and adaptable tool for bloggers.

Although I initially developed the blog template with WordPress users in mind, it is currently only available in a static page format. It is a ongoing project that I look foward to share once it's fully complete.
