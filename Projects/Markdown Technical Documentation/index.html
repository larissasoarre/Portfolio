What is Markdown?

Markdown is a markup language that is known for its simplicity and ability to convert text into HTML easily. In 2004, John Gruber and Aaron Swartz developed this language as a way to write content in a readable and straightforward format that could later be converted into HTML.


Markdown has become popular for creating web content, such as blog posts, README files, and documentation. With its straightforward syntax for headings, lists, links, and other formatting elements, it allows writers to format their content using simple, plain text without needing to write HTML code directly.

Headings

You can use the hash symbol # in Markdown to format a heading. The number of hash symbols you use at the beginning of a line indicates the heading level. For instance, to create a level-three heading <h3>, add three hash symbols before the text e.g., ### Heading.

                    
# Heading level 1
## Heading level 2
### Heading level 3
#### Heading level 4
##### Heading level 5
###### Heading level 6

Output

Heading level 1

Heading level 2

Heading level 3

Heading level 4

Heading level 5
Heading level 6

Paragraphs

To create paragraphs, use a blank line to separate one or more lines of text.

                    
I love using Markdown to jazz up my text!

I think I'll use it to create my README file.

Output

I really like using Markdown.

I think I'll use it to format my README file.

Line Breaks

To create a line break <br>, simply add two or more spaces at the end of a line and then press the return key. This will create a new line without starting a new paragraph.

                    
This is the first line.
And this is the second line.

Output

This is the first line.
And this is the second line.

Emphasis

In Markdown, you can emphasize text by making it bold or italic.

Bold

If you want to make text bold, simply enclose the word or phrase with two * or _ before and after it. You can also add emphasis to a specific part of a word by enclosing it with two asterisks without spaces around the letters. It's a straightforward way to add emphasis and highlight important text.

                        
**Boldness** is an essential quality to have.
__Bold text__ is just so great!.
When it comes to text, love and **bold**ness go hand in hand.

Output

Boldness is an essential quality to have.

Bold text is just so great!.

When it comes to text, love and boldness go hand in hand.

Italic

To italicize text, enclose the word or phrase with one * or _ before and after it. If you want to emphasize a particular part of a word, you can enclose it with one asterisk without spaces around the letters.

                        
I think *italicized* text is just the best!
_Italicized_ text is cool.
*Ital*ics make text pop.

Output

I think italicized text is just the best!

Italicized text is cool.

Italics make text pop.

Bold and Italic

To give text extra emphasis, you can make it bold and italicized at the same time by adding three asterisks or underscores before and after the word or phrase. If you want to bold and italicize a part of a word, simply add three asterisks without any spaces around the letters you want to emphasize.

                    
***Bold and italicized*** is the ultimate combo.
When in doubt, use ___bold and italicized___.
__*Bold and italicized*__ is the power move.
Two words: **_bold and italicized_**.
Never underestimate bold***and***italicized.

Output

Bold and italicized is the ultimate combo.

When in doubt, use bold and italicized.

Bold and italicized is the power move.

Two words: bold and italicized.

Never underestimate boldanditalicized.

Blockquotes

To create a blockquote, add a > in front of a paragraph.

                    
> After a long day at work, she likes to unwind by taking a hot bath and reading a good book while sipping on a cup of tea.

Output

The rendered output looks like this:

After a long day at work, she likes to unwind by taking a hot bath and reading a good book while sipping on a cup of tea.

Blockquotes with Multiple Paragraphs

To include multiple paragraphs in a blockquote, simply insert a > symbol on the blank line between each paragraph.

                    
> After a long day at work, she likes to unwind by taking a hot bath and reading a good book while sipping on a cup of tea.
>
> She finds that this routine helps her to relax and forget about the stresses of the day.

Output

The rendered output looks like this:

After a long day at work, she likes to unwind by taking a hot bath and reading a good book while sipping on a cup of tea.


She finds that this routine helps her to relax and forget about the stresses of the day.

Nested Blockquotes

Blockquotes can be nested. Add a >> in front of the paragraph you want to nest.

                    
> After a long day at work, she likes to unwind by taking a hot bath and reading a good book while sipping on a cup of tea.
>
>> She finds that this routine helps her to relax and forget about the stresses of the day.

Output

The rendered output looks like this:

After a long day at work, she likes to unwind by taking a hot bath and reading a good book while sipping on a cup of tea.

She finds that this routine helps her to relax and forget about the stresses of the day.

Blockquotes with Other Elements

Markdown-formatted elements can be included in blockquotes. However, not all elements can be used and experimentation may be necessary to determine which ones are compatible.

                    
> #### My favorite hobbies
>
> - Playing video games
> - Reading books
> - Listening to music
>
> I love spending my free time doing **fun** activities!

Output

The rendered output looks like this:

My favorite hobbies


  • Playing video games
  • Reading books
  • Listening to music

I love spending my free time doing fun activities!

Lists

You can organize items into ordered and unordered lists.

Ordered Lists

Start an ordered list by adding line items with numbers followed by periods. The order of the numbers doesn't have to be numerical, but the list should start with number one.

                    
1. Item one
2. Item two
3. Item three
4. Item four

1. Item one
7. Item two
3. Item three
9. Item four

Output

  1. Item one
  2. Item two
  3. Item three
  4. Item four

  1. Item one
  2. Item two
  3. Item three
  4. Item four

Unordered Lists

You can make an unordered list in Markdown by using dashes -, asterisks *, or plus signs + in front of line items. 

                    
- Item one 
- Item two
- Item three
- Item four

* Item one 
* Item two
* Item three
* Item four

+ Item one 
+ Item two
+ Item three
+ Item four

- Item one 
- Item two
- Item three
    - Indented item
    - Indented item
- Item four

Output

  • Item one
  • Item two
  • Item three
  • Item four

  • Item one
  • Item two
  • Item three
  • Item four

  • Item one
  • Item two
  • Item three
  • Item four

  • Item one
  • Item two
  • Item three
    • Indented item
    • Indented item
  • Item four

Code

To denote a word or phrase as code, enclose it in backticks `.

                    
You can use the `for` loop in Python to iterate over a list or other iterable object.

Output

You can use the for loop in Python to iterate over a list or other iterable object.

Escaping Backticks

Enclose a word or phrase in double backticks `` to indicate that it should be interpreted as code, even if it includes one or more backticks. This is known as escaping.

                    
``Use `code` in your Markdown file.``

Output

Use code in your Markdown file.

Code Blocks

You can create a code block by either indenting every line of the block by at least four spaces or by using one tab.

                    
function multiply(a, b) {
  return a * b;
}

console.log(multiply(5, 7));

Output

The rendered output looks like this:

                                
function multiply(a, b) {
  return a * b;
}

console.log(multiply(5, 7));

Links

To create a link, enclose the link text in brackets (e.g., [Dublin Template]) and then follow it immediately with the URL in parentheses (e.g., (https://larissasoares.com/projects/dublin%20template/)).

My favorite template is [Dublin Template](https://larissasoares.com/projects/dublin%20template/).

Output

The rendered output looks like this:


My favorite template is Dublin Template.

Adding Titles

If you want to provide additional context to a link, you can add a title by enclosing it in quotation marks after the URL. This title will appear as a tooltip when the user hovers over the link.

My favorite template is [Dublin Template](https://larissasoares.com/projects/dublin%20template/ "The best template for your blog!").

Output

The rendered output looks like this:


My favorite template is Dublin Template.

URLs and Email Addresses

By enclosing a URL or email address in angle brackets, you can convert it into a clickable link quickly.

                    
<https://www.larissasoares.com>
<email@example.com>

Output

https://www.larissasoares.com
email@example.com

Formatting Links

Links can be emphasized by adding asterisks before and after the brackets and parentheses. You can also denote links as code by adding backticks inside the brackets.

                    
I need to check something on **[Google](https://google.com)**.
This is the *[Markdown Guide](#)*.
See the section on [`code`](#code).

Output

I need to check something on Google.

This is the Markdown Guide.

See the section on code.

Images

One way to add an image in Markdown is by using an exclamation mark ! followed by the alt text enclosed in square brackets, and the path or URL of the image enclosed in parentheses. A title can also be added as an optional attribute enclosed in quotation marks after the path or URL.

Escaping Characters

Escape special characters in a Markdown document by placing a backslash \ in front of the character to prevent it from being interpreted as formatting syntax.

\* Without the backslash, this would be a bullet in an unordered list.

Output

* Without the backslash, this would be a bullet in an unordered list.

Characters You Can Escape

You can use a backslash to escape the following characters.

Character Name
\ backslash
` backtick
* asterisk
_ underscore
{ } curly braces
[ ] brackets
< > angle brackets
( ) parentheses
# pound sign
+ plus sign
- minus sign (hyphen)
. dot
! exclamation mark
| pipe

HTML

You can use HTML tags in Markdown-formatted text in many Markdown applications. This feature comes in handy if you prefer specific HTML tags to Markdown syntax, such as using HTML tags for images, headings, code, and so on. It's also useful when you need to modify an element's attributes, such as changing the text color or adjusting an image's width.

To incorporate HTML tags into your Markdown-formatted file, simply include them in the text.

This `word` is inline code. This <h1>word</h1> is a level-one heading.

Output

The rendered output looks like this:


This word is inline code. This

word

is a level-two heading.
